# Day 03 – Linux Commands Practice

Today I practiced some important Linux commands used for process management, file handling, and networking troubleshooting.

---

# Process Management Commands

| Command | Usage |
|---|---|
| `ps aux` | Show all running processes |
| `top` | Monitor running processes live |
| `htop` | Interactive process viewer |
| `kill PID` | Stop a process |
| `kill -9 PID` | Force kill a process |
| `pgrep nginx` | Find process ID by name |
| `jobs` | Show background jobs |
| `bg` | Run job in background |
| `fg` | Bring job to foreground |
| `uptime` | Show system uptime |

---

# File System Commands

| Command | Usage |
|---|---|
| `pwd` | Show current directory |
| `ls -la` | List files with details |
| `cd` | Change directory |
| `mkdir demo` | Create a directory |
| `rm -rf demo` | Remove directory/files |
| `cp file1 file2` | Copy files |
| `mv file1 file2` | Move or rename files |
| `touch notes.txt` | Create empty file |
| `cat notes.txt` | View file content |
| `find / -name test.txt` | Search for files |
| `df -h` | Show disk space usage |
| `du -sh folder` | Show folder size |

---

# Networking Commands

| Command | Usage |
|---|---|
| `ping google.com` | Check internet connectivity |
| `ip addr` | Show IP address details |
| `curl google.com` | Test website response |
| `dig google.com` | Check DNS information |
| `ss -tulnp` | Show open ports and sockets |
